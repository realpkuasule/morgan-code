📝 保存当前版本
   ./vc save "描述你的更改"

📋 查看所有版本
   ./vc list

📊 查看当前状态
   ./vc status

🔄 回滚到指定版本
   ./vc rollback <版本标签>

🔍 查看版本详情
   ./vc show <版本标签>

📊 比较两个版本
   ./vc diff <版本1> <版本2>

📦 创建 tar.gz 备份
   ./vc backup

❓ 显示帮助
   ./vc help

💡 常用场景

1. 开发前保存：
   ./vc save "开始开发新功能前的稳定版本"

2. 功能完成后保存：
   ./vc save "完成XXX功能"

3. 出问题时回滚：
   ./vc list                    # 查看历史版本
   ./vc rollback v20260310-150000  # 回滚

4. 查看改了什么：
   ./vc diff v20260310-150000 v20260311-062045

🔒 安全特性

✓ 回滚前自动创建备份分支
✓ 需要输入 'yes' 确认回滚操作
✓ 完整的 Git 历史记录
✓ 支持恢复到任意历史版本
